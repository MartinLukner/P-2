Fuzz tester for Augmented Backus-Naur Form (ABNF) rules as defined in RFC 5234. This library can directly parse the ABNF rules, then generate random output matching those rules. That output can be fed in a test framework such as JUnit to search for defects in an application which implements the rules. The most common use case would be testing implementations of other IETF RFCs.

This library does not support the original BNF format. However you can manually update the syntax to match.

In order to use the library you must first create an input file containing ABNF rule definitions. In most cases you can do this by copying and pasting rules from a larger specification document such as an IETF RFC. Create a Fuzzer object from that file, and then call one of the generate methods to create each test value. For example, if you want to test an implementation of an ABNF rule named "foo" from RFC "9999" the code would look something like this.

File file = new File("rfc9999.txt");
Fuzzer fuzzer = new Fuzzer(file);
System.out.println(fuzzer.generateAscii("foo"));
